General

Yellow fields are editable. As the BEW and Basic Arm changes over time, those values have to be taken from the actual Weight And Balance sheet in the airplane, to be inserted in first row.

Passenger Details

  1. Enter total passenger weight in kg.
  2. Enter total number of passengers.
  3. Under Seating, enter number of passengers for each row.
  4. For removing or installing seats, tick or untick in appropriate row under single
    or double, and weights will be automatically added or subtracted accordingly.
  5. If more accurate weights are needed, tick for individual passenger weights,
    and enter total weight in kg for each row.

Cargo Details

  1. Enter weight in kg. for each zone or pod.
  2. If any special load, enter the cargo weight, without special load,
    and the special load will automatically be added.

Special Load

  1. Tick/untick for each item onboard/not onboard.
  2. Enter actual weight.
  3. The weight will be added to appropriate cargo zone or cargo pod,
    and distributed as described in the Loading coloumn.
