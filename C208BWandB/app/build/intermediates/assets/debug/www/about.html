General info

This application is for calculating weight and balance for the Cessna Caravan C208B, with cargo pod and payload extender kit installed.

The seat configuration is company specific, and in this configuration, the seat row 4 is normally not used. The special load is also specific for the company this application is intended for.

Disclaimer

This application is a third party application, and not an official Cessna certified application. It is used purely on the user's own responsibility, and only to be used as a supplement to Cessna's official manuals.
